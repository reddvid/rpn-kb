    Scrolling Text in XSplit

    On this page
    • Pre-Requisites
    • Guide
    • Getting Help

    This will show you how to edit and update scroll text in XSplit by editing a .txt file instead of editing inside the software

    Pre-Requisites#

    This guide assumes that you already have scroll text in XSplit.

    Guide#

    1. Create a new Text Document (.txt) somewhere secure in you desktop and name it appropriately (e.g. scrolltext.txt)

    2. Open XSplit Broadcaster

    3. On your current scene/stage, open the Scroll Text source Properties by right-clicking

    4. Check Use Custom Script box which enables the Edit Script button

    Use the Script tab if you instead copied the code

    1. Click Edit Script button. This will open the Edit Script - Text Window

    2. Choose Load Text from Local File as Template from the dropdown menu

    1. In the File Path box, click the ellipsis to browse your .txt file you created in Step 1. Keep other settings as is but set Ignore Line Breaks. Finally click Update Text to save settings.

    1. It should now load the contents from the .txt file. Otherwise, try saving again on the .txt file or Click Update Text again on the Edit Script window.

    2. Copy the scroll text source to your other scene(s).

    3. You should now be able to edit the Scroll Text by editing the .txt file.

    Getting Help#

    Contact IT admin thru email or Viber.
