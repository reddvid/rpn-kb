    Layout Standards#

    Title Standards#

    • Font: Lato Black
    • Color: Black

    Subtitle Standards#

    • Font: Lato Black
    • Color: Red

    Date Stamp Standards#

    • Format: MMM DD, YYYY

    • OCT 8, 2023

    • Font: Lato Bold

    • Color: White

    Time Stamp Standards#

    • Format: HH: MM TT

    • 04:12 PM

    • DO NOT include seconds

    • Font: Lato Bold

    • Color: White

    Scroll Text Standards#

    • Include daily headline news, announcements, breaking news, etc.

    • Include hotlines, emergency hotline

    • Include social media usernames

      • Facebook Page username
      • Twitter @ handle
      • YouTube channel

    • Use ALL CAPS

    • DO NOT include links - viewers can NOT click

    • DO NOT use too much symbols to separate text

    • Font: Lato Bold

    • Color: White

    Other Important Notes#

    • Ensure camera quality and position
    • Check lighting
    • Backdrop/Tarp should be smooth (no crease/wrinkles)

    New Standards for 2024#

    • Remove sidebar
      • No ads
      • No social media
      • No other logos
